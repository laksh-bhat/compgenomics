
All binaries (java classes) reside in bin directory. 
Resources (test input and out files) can be found in res directory.
Java source files are present in the src/asgmt5 directory

To run a program, go to the bin/production/compgenomics directory and fire commands like the following.
java -cp . asgmt5.OrfProteinCoder < ../../../res/asgmt5/rosalind_orf.txt >  ../../../res/asgmt5/rosalind_orf_ans.txt

The written answers and summary can be found under res/asgmt5 in file analysis.txt.
